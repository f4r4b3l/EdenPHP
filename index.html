EdenPHP by EdenPHP

Framework PHP ayant pour but la simplicité d'utilisation et l'accélération maximale du développement web.

Documentation en cours d'écriture

Dependencies

PHP 5.2, MySQL, Apache 2.x (Module rewrite)

Install

Actuellement : Copier simplement les fichiers dans votre répertoire source. Modifiez le fichier /config.php selon les instructions s'y trouvant. Effectuez un chmod 777 sur les dossiers suivants :

  • /cache/
  • /webroot/file/
  • /lang/
  • /log/

Bientôt : Téléchargez le fichier install.php, remplissez le formulaire et c'est tout. La dernière version des fichiers sont téléchargés depuis Github et les configurations sont automatiques.
